# MingOS Project

MingOS is the first flagship implementation of Ming Foundation standards.

## Role

MingOS turns the Foundation into usable infrastructure and applications for continuous understanding of life, relationship, events, state, memory, reflection, choice, action, and growth.

## MingOS is not

- the whole Ming Foundation;
- a single family education product;
- a chatbot with a custom prompt;
- a system for controlling children or families;
- a diagnostic substitute;
- an authority over personal life decisions.

## Initial implementation surfaces

- `mingos.cn` — public brand, charter, trust, and participation portal;
- enterprise WeChat — natural conversation and timely follow-up channel;
- H5 — identity binding, confirmation, lightweight actions, and reports;
- `ymai.love` — evolving full MingOS application;
- admin / human-support workbench — authorized review and intervention.

## Architectural requirement

All surfaces must use one shared MingOS Core for:

- identity;
- family and relationship graph;
- consent;
- life events;
- observation;
- memory;
- state;
- action and feedback;
- audit.

Channels must not become separate sources of truth.

## First reference scenario

Parent-child conflict understanding and repair:

```text
WeChat expression
  → identity and family resolution
  → life event
  → revisable observation
  → H5 confirmation
  → user-chosen small action
  → follow-up
  → feedback
  → memory and state update
  → weekly reflection
```
