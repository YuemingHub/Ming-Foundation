# Foundation 1.0 Delivery Roadmap

## Milestone 1 — Repository Foundation

Acceptance:

- repository architecture exists;
- metadata and templates are defined;
- validation runs in CI;
- mission, vision, principles, and standards lifecycle are committed;
- first ADRs are accepted.

## Milestone 2 — Ontology Foundation

Target documents:

- MOS-0001 First Principles;
- MOS-0002 Life Ontology;
- MOS-0003 Identity;
- MOS-0004 Relationship;
- MOS-0005 Life Event;
- MOS-0006 State;
- MOS-0007 Context;
- MOS-0008 Consent.

## Milestone 3 — Observation and Memory

Target documents:

- Observation Protocol;
- Knowledge Status Model;
- Evidence and Confidence;
- Memory Layers;
- Revision and Forgetting;
- Timeline and Pattern rules.

## Milestone 4 — MingOS Core Contract

Target outputs:

- canonical entity schemas;
- API boundaries;
- event contracts;
- audit model;
- channel identity binding;
- reference implementation architecture.

## Milestone 5 — First Complete User Loop

Reference scenario:

- enterprise WeChat expression;
- H5 confirmation;
- user-chosen action;
- follow-up;
- feedback;
- weekly reflection.

## Milestone 6 — Foundation 1.0 Candidate

Release requires:

- working reference implementation;
- documented case validation;
- safety and ethics review;
- migration guidance;
- public limitations;
- no unresolved Charter conflict.
