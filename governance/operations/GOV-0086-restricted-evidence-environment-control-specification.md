# GOV-0086 — Restricted Evidence Environment Control Specification

## Required live controls

1. restricted evidence workspace;
2. role-based access control;
3. contact and evidence separation;
4. enforceable retention and deletion;
5. incident, stop, and access logging;
6. export and copy controls;
7. backup restoration suppression;
8. withdrawal and deletion verification.

## Synthetic environment

Day 16 uses repository-only synthetic fixtures.

It MUST contain no real person, family, child, practitioner, client,
third-party, crisis, medical, legal, school, financial, immigration, or
safeguarding evidence.

## State

```text
Control designs complete: 8
Synthetic controls enforced: 8
Live controls ready: 0
Live environment: NotProvisioned
Live evidence ingestion: Prohibited
```

A complete design is not a provisioned evidence environment.
