# REF-0020 — Named Accountability and Human Activation Guide

```text
Role defined
≠ person nominated

Person nominated
≠ role accepted

Role accepted
≠ qualification verified

Qualification verified
≠ conflict-free

Public role code
≠ public real name

Protocol route defined
≠ human use approved

Environment plan complete
≠ environment deployed

Accountability tabletop passed
≠ CP2 authorized
≠ CP3 authorized
```

The correct next step after Day 17 is restricted nomination and real
verification, not public naming or participant recruitment.
