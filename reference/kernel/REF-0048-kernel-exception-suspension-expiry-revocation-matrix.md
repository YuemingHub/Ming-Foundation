# REF-0048 — Kernel Exception Suspension Expiry and Revocation Matrix

| Event | Minimum consequence | Required record |
|---|---|---|
| Exception proposed | no authority until review | scope, basis, safeguards, owner |
| Exception expires | stop affected authorization | expiry and dependency trace |
| Serious harm evidence | immediate claim suspension | incident and containment |
| Evidence invalidated | narrow or suspend claim | invalidation and reassessment |
| Material change | affected results invalidated | change and regression map |
| Deceptive wording | correction and suspension review | claim audit and remediation |
| Review conflict | replace or qualify reviewer | conflict record |
| Remediation fails | revoke or continue suspension | accountable decision |
