# REF-0051 — Kernel Public Claim and Mark Language Matrix

| Wording | Current status | Qualification |
|---|---|---|
| implements selected Draft Kernel concepts | permitted with evidence | exact concepts, version, limitations |
| under internal review against Draft KERNEL-* | permitted | boundary, date, no approval implication |
| Kernel-ready | prohibited/ambiguous | do not use |
| MingOS Kernel conforming | prohibited now | unavailable while documents are Draft |
| MingOS certified/approved | prohibited now | no authority exists |
| Charter compliant | prohibited now | Candidate Charters and no accepted method |
| badge, seal or official mark | prohibited now | requires separate accepted decision |
