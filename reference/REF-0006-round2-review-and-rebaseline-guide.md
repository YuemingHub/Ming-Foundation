# REF-0006 — Round 2 Review and Requirement Rebaseline Guide

## Distinctions

```text
Source marker test passed
≠ product behavior tested

Internally accepted ambiguity answer
≠ ambiguity closed

Historical requirement index retained
≠ complete index for revised source

Round 2 completed
≠ external or affected-person review completed

RFC internally acceptable with residuals
≠ Candidate
```

## Re-baseline sequence

```text
Revised RFC source
  ↓
Enumerate every normative clause
  ↓
Map legacy requirement IDs
  ↓
Create new or superseding requirement IDs
  ↓
Update implementation acceptance specifications
  ↓
Validate source and machine consistency
  ↓
Internal re-review
  ↓
Affected-person, domain, legal, and external review
  ↓
Separate promotion decision
```
