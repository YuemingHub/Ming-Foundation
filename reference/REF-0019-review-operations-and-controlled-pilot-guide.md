# REF-0019 — Review Operations and Controlled Pilot Guide

```text
Role defined
≠ person assigned

Protocol designed
≠ human use approved

Environment specified
≠ environment provisioned

Synthetic pilot passed
≠ affected-person review complete

Repository test passed
≠ product conformance
```

Day 16 permits learning about the control architecture without exposing real
people or evidence to an unactivated process.
