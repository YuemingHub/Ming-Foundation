# REF-0035 — Restricted Nomination and CP2 Pre-Authorization Guide

```text
Nomination slot
≠ real nomination

Real nomination
≠ voluntary acceptance

Acceptance
≠ qualification

Qualification
≠ conflict-free independence

Private workspace exists
≠ controls deployed

Protocol applicability defined
≠ protocol approved

Conditional pre-authorization
≠ effective authorization

CP2 authorization
≠ CP3 authorization
```

The correct Day 18 outcome is a complete and bounded route to activation,
while CP2 remains blocked.
